#  Семантический поиск по вакансиям

Веб-приложение для поиска вакансий по смыслу, а не по ключевым словам. Использует мультиязычную модель для преобразования текста в векторы и FAISS для быстрого поиска ближайших соседей.

## Возможности

- **Семантический поиск** - находит вакансии по смыслу, даже если нет точного совпадения слов
- **Мультиязычность** - поддерживает русский и английский языки
- **Веб-интерфейс** - удобный интерфейс на Gradio
- **Масштабируемость** - FAISS индекс позволяет быстро искать среди тысяч вакансий

## Технологии

- **Python 3.8+**
- **Sentence Transformers** - для генерации эмбеддингов
- **FAISS** - для быстрого поиска по векторам
- **SQLite** - для хранения вакансий
- **Gradio** - для веб-интерфейса

## Установка

1. Клонируйте репозиторий:

2. Создайте виртуальное окружение:

## Использование

### 1. Генерация тестовых данных
```bash
cd scripts
python create_vacancy_db.py
```
Создаст базу данных с 200 синтетическими вакансиями.

### 2. Построение индекса
```bash
python build_index.py
```
Преобразует вакансии в эмбеддинги и строит FAISS индекс.

### 3. Поиск в терминале
```bash
python search.py
```
Интерактивный поиск в командной строке.

### 4. Веб-интерфейс
```bash
python app.py
```
Запустит веб-сервер на http://localhost:7860

## Конфигурация

Основные параметры в файлах:

**create_vacancy_db.py**
- `NUM_VACANCIES` - количество вакансий (по умолчанию 200)
- `USE_RUSSIAN` - язык описаний (True - русский, False - английский)

**search.py / app.py**
- `TOP_K` - количество результатов поиска (по умолчанию 5)
- `MODEL_NAME` - модель Sentence Transformer

## Как это работает

1. **Подготовка данных**: Вакансии из БД преобразуются в эмбеддинги (векторы) с помощью SBERT
2. **Индексация**: Эмбеддинги индексируются в FAISS для быстрого поиска
3. **Поиск**: Запрос пользователя тоже превращается в эмбеддинг
4. **Ранжирование**: FAISS находит ближайшие векторы вакансий
5. **Результат**: Система возвращает вакансии с наибольшим семантическим сходством

## Логирование

- Создание БД
- Построение индекса
